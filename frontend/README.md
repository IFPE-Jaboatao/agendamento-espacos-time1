# 🎨 Agendamento de Espaços - Frontend

Interface moderna e responsiva para o sistema de agendamento, permitindo que usuários solicitem reservas e administradores gerenciem o campus.

## 🛠️ Tecnologias
- **React.js** com **TypeScript**
- **Vite** (Build Tool)
- **Tailwind CSS** (Estilização)
- **Flowbite React** (Componentes de UI)
- **Axios** (Consumo de API)
- **React Router Dom** (Navegação)

## 🚀 Funcionalidades
- **Login/Logout**: Controle de sessão via JWT.
- **Dashboard**: Visão geral de status.
- **Gestão de Espaços**: Cadastro e edição (Admin).
- **Reservas**: Solicitação e acompanhamento de status.

## ⚙️ Como Rodar
1. Instale as dependências: `npm install`
2. Certifique-se de que a API (Backend) está rodando em `http://localhost:3000`.
3. Inicie o servidor de desenvolvimento: `npm run dev`
4. Acesse: `http://localhost:5173`

## 🔑 Perfis de Acesso
- **ADMIN**: Pode gerenciar usuários, criar espaços e aprovar reservas.
- **USUÁRIO**: Pode visualizar espaços e solicitar agendamentos.
